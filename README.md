# Domino用 エレクトーン 音源定義ファイル

## 使用方法

1. [electone.xml](./electone.xml)を`Domino.exe`と同じ位置にある`Module`フォルダ内にコピーする。
1. Dominoを起動(再起動)する。
1. 「ファイル」→「環境設定」→「MIDI-OUT」、任意のポートの音源を`Electone`に変える
